# JWT viewer

## Backend

NestJS / Sqlite / TypeORM

git clone https://github.com/iromashko/jwt-backend.git \
cd jwt-backend\
npm i \
npm run start

## Frontend

Angular

git clone https://github.com/iromashko/jwt-frontend.git \
cd jwt-frontend \
npm i \
ng s

## Testing

Cypress

cd jwt-frontend \
./node_modules/cypress/bin/cypress open
